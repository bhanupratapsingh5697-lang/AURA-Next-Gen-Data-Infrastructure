# AURA-Next-Gen-Data-Infrastructure

A premium, highly interactive frontend web application showcasing advanced web development techniques. 

## Features
- **Custom WebGL Shaders**: Liquid chrome/mercury morphing blob built with Three.js vertex and fragment shaders.
- **Advanced GSAP Animations**: Custom staggered text-splitting, magnetic buttons, and multi-layer parallax scrolling.
- **Scroll Hijacking**: Horizontal scroll pinning implemented with Lenis smooth scroll and ScrollTrigger.
- **High-End UI/UX**: Custom SVG geometric cursors, film grain overlays, and glassmorphism.

## Tech Stack
- HTML5 / CSS3
- JavaScript (ES6+)
- Three.js
- GSAP (ScrollTrigger)
- Lenis (Smooth Scroll)
- Vite
